# rest_api_ns

#App Create using spring boot

To deploy only need the application jar and JDK 8+
Then run `java -jar rest-api-ns-0.0.1-SNAPSHOT.jar`

#Hal Implementation

You can use HAL by default going to http://localhost:8080/
